# soupbot
A multifunctional discord bot. 

### Capabilities
- Can stream audio from multiple sources into voice calls using [Lavalink](https://github.com/lavalink-devs/Lavalink).
- Uses the Steam web API to pull live game player counts.
- Able to translate text using DeepL and Microsoft Azure Translator.
- Various other fun commands for users to interact with such as asking a magic 8 ball questions, or getting a daily fortune.
- Utilizes a small database to store some basic user data, preferences, and Steam app data.
- Supports both legacy and slash command methods of interacting with the bot.
